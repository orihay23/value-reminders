# Value Reminders

A daily message app that sends you curated content — scripture, quotes, and lyrics — filtered through your 5 personal values.

## What It Does

Value Reminders delivers a daily dose of meaning straight to you. Each day you receive hand-picked content drawn from:

- **Scripture** — timeless wisdom from sacred texts
- **Quotes** — words from thinkers, leaders, and creators
- **Lyrics** — lines from songs that speak to the soul

Every piece of content is filtered through the **5 values you define**, so what arrives in your inbox always feels personal and intentional — not generic.

## How It Works

1. Set your 5 core values (e.g., courage, faith, family, growth, integrity)
2. The app curates daily content that aligns with those values
3. You receive a message each day with something worth reflecting on

## Getting Started

Upload the zip file, extract it, and open `index.html` in your browser — or deploy to GitHub Pages for a live link you can share.
